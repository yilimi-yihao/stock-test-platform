【发散思维和联想关联规则】
必须考虑以下维度（即便关联较弱，也要列出，用 relevance 区分强弱）：
- 谐音效应：七夕 -> 鹊桥相关、520 -> 珠宝/化妆品、666 -> 游戏/彩票
- 情绪传导：节前消费情绪高涨、节后复工效应（交通/制造业）
- 反向影响：长假期间A股休市，但港股/美股波动影响节后开盘
- 行业周期：高考 -> 教育培训/文具、毕业季 -> 招聘/租房/家电、冬至 -> 羊肉/暖冬概念
- 情感经济：情人节 -> 珠宝/鲜花/巧克力（食品）、母亲节 -> 化妆品/家居
- 仪式感消费：春节 -> 礼品/白酒/影视、七夕 -> 钻石/珠宝

2. 单日节日（如双十一）holiday_date 填当天，is_trading_closed 填 false

   - "stock": 个股，code 填6位代码
   优先使用 "industry" 类型以便关联股票数据库
5. relevance 字段（浮点数 0.0~1.0）评分标准：
   - 0.9~1.0：历史数据明确印证的强相关（如春节-白酒、国庆-旅游）
   - 0.7~0.9：逻辑清晰的较强相关（如世界杯-啤酒）
   - 0.5~0.7：中等相关，有合理逻辑支撑
   - 0.3~0.5：弱相关，需要一定联想
   - 0.1~0.3：谐音/情绪/联想类，较发散但有一定市场认知

   - direct：直接受益的板块
   - indirect：间接关联
   - phonetic：谐音/情感联想类（relevance 通常 0.1~0.4）
